# ejercicio_py

## 📘 Descripción del proyecto

**ejercicio_py** es un repositorio que contiene una actividad práctica desarrollada en **Python** para aplicar conceptos básicos de **Programación Orientada a Objetos (POO)**.

El proyecto está compuesto por **cuatro ejercicios**, donde se utilizan clases, objetos, atributos y métodos para representar diferentes situaciones.

---

## 🧠 Conceptos aplicados

En estos ejercicios se practican los siguientes temas:

* Creación de clases
* Uso de objetos
* Definición de atributos
* Creación de métodos
* Uso de estructuras condicionales (`if`)
* Manipulación de datos dentro de objetos

---

## 📂 Ejercicios incluidos

### 1️⃣ Ejercicio: Libro

Clase `Libro` que permite representar un libro con su información básica.

Funciones:

* Guardar título, autor y número de páginas.
* Mostrar la información del libro.
* Modificar el número de páginas.

---

### 2️⃣ Ejercicio: Producto

Clase `Producto` que simula un pequeño sistema de inventario.

Funciones:

* Verificar disponibilidad de productos.
* Realizar ventas.
* Restablecer o aumentar el stock.

---

### 3️⃣ Ejercicio: Vehículo

Clase `Vehiculo` que representa un automóvil.

Funciones:

* Mostrar información del vehículo.
* Acelerar.
* Frenar.
* Verificar si el vehículo supera un límite de velocidad.

---

### 4️⃣ Ejercicio: Estudiante_nota

Clase `Estudiante` que permite gestionar la información académica de un estudiante.

Funciones:

* Guardar nombre del estudiante.
* Registrar notas.
* Calcular o mostrar resultados relacionados con las calificaciones.

---

## ▶️ Cómo ejecutar el proyecto

1. Tener **Python** instalado en el sistema.
2. Descargar o clonar este repositorio.
3. Ejecutar el archivo del ejercicio que desees probar.

Ejemplo:

```bash id="5l4c8h"
python vehiculo.py
```
